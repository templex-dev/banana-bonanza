Banana Bonanza (Alpha) uses procedurally generated pixel sprites and tiles at runtime,
so this folder stays light. The engine creates:
- 16×16 tile atlas (grass/stone/mud/water/bridge/vines + markers)
- ~24px-tall characters (monkey, gorilla, minions)
- Banana pickups (normal/golden), dust/hit FX, etc.
All graphics use nearest-neighbor & integer snapping for crisp pixels.
