# banana-bonanza

a fast-paced 2d pixel raid where you and your friends try to snatch bananas from an angry gorilla.  
play solo, co-op, or go against each other.

## features
- quick matches with simple controls
- solo, co-op, and competitive modes
- chaotic gorilla ai that's always watching
- pixel art style

## how to play
1. sneak past the gorilla
2. grab as many bananas as you can
3. escape before time runs out (or before the gorilla catches you)
